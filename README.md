# UBER-Supply-Demand
The project analyzes Uber request data to study customer demand &amp; service outcomes. The data is explored using different visualizations based on trip status, pickup point, request time, &amp; drop time. The goal is to understand where Uber is performing well &amp; where service issues such as trip cancellations &amp; cab unavailability are affecting operations

Overall, the project helps identify service gaps and provides useful insights to improve operational efficiency and customer satisfaction.
